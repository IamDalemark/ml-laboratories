## ğŸš€ Getting Started

### Prerequisites

- Python 3.x
- pip package manager

### Installation

1. Clone this repository:

   ```bash
   git clone https://github.com/IamDalemark/ml-laboratories.git
   cd ML-laboratories
   ```

2. Create and activate a virtual environment (recommended):

   **On macOS/Linux:**

   ```bash
   python3 -m venv .venv
   source .venv/bin/activate
   ```

   **On Windows:**

   ```bash
   python -m venv .venv
   .venv\Scripts\activate
   ```

3. Install required dependencies:

   ```bash
   pip install -r requirements.txt
   ```

   To deactivate the virtual environment when you're done:

   ```bash
   deactivate
   ```

### Running the Notebooks

Launch Jupyter Notebook or JupyterLab:

```bash
jupyter notebook
```

Then navigate to the desired lab folder and open the `.ipynb` files.
